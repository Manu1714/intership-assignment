1. What is a Component?

A component is a reusable, independent building block of a webpage. Every website is made up of multiple components such as Navbar, Hero Section, Cards, Footer, etc. Breaking a website into components helps developers build, manage, and reuse parts easily.

2. Website Chosen: Amazon.in

Amazon is one of the most popular e-commerce websites in the world. It has a rich UI with many reusable components.

3. Components Identified

No. Component Name Description
1 Header / Top Bar Logo, search bar, sign-in button, cart icon
2 Navigation Bar Category links like Electronics, Fashion, Home, etc.
3 Hero / Banner Section Large sliding image banners with offers and deals
4 Product Card Image, title, price, rating, add to cart button
5 Product Grid / List Multiple product cards arranged in rows and columns
6 Deals Section Today's deals with countdown timer and discount badges
7 Category Section Icons or images for each product category
8 Sidebar / Filters Filter by brand, price range, rating, etc.
9 Footer Links, policies, social media icons, copyright

Breaking a website like Amazon into components shows how modern websites are structured. Each component is independent, reusable, and has its own purpose. In React or any modern framework, each of these components would be a separate file, making the code easy to manage and reuse.
